一个**轻量、飞快、美观**的文档生成的工具，你可以利用它为你的项目生成文档。

[English](./README_EN.md) | 简体中文

## Features

- **`Vite`** 内核，高效极速的开发体验
- 使用 `Markdown` 写法，让你快速为项目编写说明文档的同时，自动生成预览，在调试的同时生成文档
- 自动分析 `TypeScript` 类型定义并生成 API 文档
- 自动根据 `.md` 文件目录生成菜单导航

## Usage & Guide

你可以查看 `examples` 目录下的项目，便于你快速开始使用。

```bash
$ npm install -g vitdoc
$ cd ../examples/demo
$ vitdoc 
```
